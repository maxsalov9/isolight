# IsoLight — Unity Prototype Starter Pack

This pack contains the core planning and Codex instruction files for the first IsoLight Unity prototype.

## Project

**Name:** IsoLight  
**Genre:** Isometric post-apocalyptic RPG / tactical action RPG  
**Tagline:** Every light is an isle.

## How to use

1. Create a new Unity project named `IsoLight`.
2. Copy the `Docs/IsoLight` folder into the root of the Unity project.
3. Copy `.gitignore` and `.gitattributes` into the root of the Unity project.
4. Initialize Git and push the project to GitHub.
5. Connect the repository to Codex.
6. Give Codex the prompt from `FIRST_CODEX_PROMPT_BATCH_1.md`.
